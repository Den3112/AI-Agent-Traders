# Contributing to AI Agent Traders

Thank you for your interest in contributing to AI Agent Traders! We welcome contributions from the community to help make this the best autonomous trading platform.

## How Can I Contribute?

### Reporting Bugs

- Use the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md).
- Provide a clear description and steps to reproduce.

### Suggesting Enhancements

- Use the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).
- Explain why the feature would be useful.

### Pull Requests

1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add some amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

## Development Setup

Refer to the [README.md](README.md) for environment setup instructions.

## Code Style

- Follow PEP 8 for Python.
- Use Prettier for JavaScript/JSON.
- Ensure all tests pass before submitting.
