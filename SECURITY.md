# Security Policy

## Supported Versions

Currently, only the latest version of AI Agent Traders is supported.

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, report it privately.

Please send an email to [security@example.com](mailto:security@example.com) (replace with actual email if available) with a description of the vulnerability and steps to reproduce.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.
