# Правила для AI-агентов Dashboard

Этот раздел содержит специфические требования к коду и дизайну фронтенда.

## Визуальный "Золотой стандарт"

1. **Цветовая палитра**:
   - Фон: Элегантный темный (`#0d0d0f`).
   - Акценты: Глубокий синий или изумрудный (`#00a8ff` / `#00ff88`).
   - Текст: Мягкий белый (`#e0e0e0`).

2. **Элементы Glassmorphism**:
   - Используйте `backdrop-filter: blur(10px)` для карточек.
   - Тонкие границы (`border: 1px solid rgba(255, 255, 255, 0.1)`).

3. **Анимации**:
   - Плавные появления (`opacity`, `transform`).
   - Интерактивные состояния кнопок (hover, active).

## Кодовая база Next.js

- Избегайте использования библиотек UI (MUI, Shadcn), если это не согласовано. Мы строим кастомный Premium UI.
- Все стили должны быть изолированы в `.module.css` или глобальных переменных.
- Обязательная проверка адаптивности (mobile-first).
