# Правила разработки Dashboard

Этот проект использует **Next.js 15+** и **React 19**. Многие соглашения отличаются от старых версий.

## Стек и правила
- **Язык**: TypeScript (строгий режим).
- **Стили**: Только Vanilla CSS. Используйте CSS Variables для темы.
- **Компоненты**: По умолчанию Server Components. Используйте `'use client'` только для интерактивности.
- **Данные**: Используйте Server Actions для мутаций.
- **Дизайн**: Следуйте гайду Premium UI в `.ai-instructions.md`.

## Команды
- `npm run dev`: Запуск сервера разработки.
- `npm run build`: Сборка проекта.
- `npm run lint`: Проверка кода.
