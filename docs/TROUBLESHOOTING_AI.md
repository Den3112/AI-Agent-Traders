# 🛠 Устранение неполадок AI Trade Team

Если вы видите, что ИИ-агенты работают нестабильно или логи заполнены ошибками, следуйте этой инструкции.

## 1. Ошибка 429: Rate Limit Reached (Gemini)

**Причина**: Превышена квота бесплатных запросов к Google AI Studio.

**Что делать**:

- Подождите 1-5 минут (лимиты обычно сбрасываются быстро).
- Переключитесь на использование `GEMINI_API_KEY` вместо `GOOGLE_API_KEY` в `.env`.
- Используйте альтернативного провайдера (через OpenRouter) в файле `openclaw.json`.

## 2. Ошибка 404: Telegram Not Found

**Причина**: В `.env` установлен текст `your_telegram_bot_token` вместо реального токена.

**Что делать**:

- Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
- Замените значение `TELEGRAM_BOT_TOKEN` в файле `.env`.
- Если бот не нужен, закомментируйте строку `TELEGRAM_BOT_TOKEN` символом `#`.

## 3. Агенты не запускаются

**Что делать**:

- Запустите `python3 scripts/ai/validate_system.py` для проверки структуры.
- Запустите `python3 scripts/ai/check_health.py` для анализа логов.
- Проверьте, запущен ли Gateway: `ps aux | grep openclaw`.

## 4. Как сменить модель для всех агентов

В файле `openclaw.json` измените значение:

```json
"agents": {
  "defaults": {
    "model": "google/gemini-1.5-pro" 
  }
}
```

*Примечание: gemini-1.5-pro обычно имеет более высокие квоты, чем flash-версия.*
